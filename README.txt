For creating a super user:
1. python manage.py createsuperuser , then fill in required details
2. MAKE SURE TO GO TO ADMIN PANEL AND REGISTER A CUSTOMER FOR SUPERUSER!


For sending mails:
1. Save emailid and password as environment variables.
2. If required, enable sending mails by less secure apps. (For more info, open settings.py and scroll to last lines)


For viewing products with no image or certificate, open www.example.com/addImage, then click on Update to add.

MAKE SURE THAT COLOR, CLARITY AND LAB ARE STANDARD(FROM THE LIST)
